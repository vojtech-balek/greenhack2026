# renovuj.me

Full-stack starter structure for the `renovuj.me` website template.

## Structure

```text
backend/
  server.mjs
frontend/
  index.html
  styles.css
  app.js
package.json
```

## Run

```bash
npm start
```

Open `http://localhost:3000`.

For development with Node's file watcher:

```bash
npm run dev
```
